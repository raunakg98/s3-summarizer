S3 Summariser

A serverless app using AWS Lambda, Bedrock, and S3 to summarize text.

How it works

  • Upload a .txt file to the raw/ folder in S3
  • Lambda is triggered automatically
  • The file is read and summarized using Titan on Bedrock
  • A summary is saved to the summary/ folder in the same bucket

API Usage

You can also call the API directly using a public URL to a text file:

POST [your-api-url] { "url": "https://www.gutenberg.org/cache/epub/11/pg11.txt" }

Tools Used

  • AWS Lambda (Python)
  • Amazon Bedrock (Titan G1 Lite)
  • Amazon S3
  • AWS SAM for deployment

Limitations

  • 4096 token input limit with Titan model
  • Large files may be truncated
  • Summaries overwrite existing files if names repeat
