About this demo

      

        This app lets you upload a `.txt` file and returns exactly five bullet points summarizing the content.
        It uses a serverless Python API with AWS Lambda + Amazon Bedrock and supports two models:
        Claude 3 Haiku and Amazon Titan.
      

      
    
        
  • Fast + lightweight serverless architecture (SAM, Lambda, S3).
    • 
        
  • Two LLM options: Claude 3 Haiku (fast/cheap) and Titan (backup).
    • 
        
  • Chunking used to handle long texts with model-friendly limits.
    • 
        
  • Limitations: Plain text only, no PDF/HTML. Cost from your Bedrock account.
    •
