S3 Summariser

Upload a text file and get a five-bullet summary using AWS Lambda & Bedrock.

1. Upload a .txt file

2. API endpoint (curl)

You can also call the API directly with a public URL:

POST https://https://obccuc1q2c.execute-api.us-east-1.amazonaws.com/summarise { "url":"https://www.gutenberg.org/cache/epub/11/pg11.txt" }

How it works

  • The page reads your file in the browser.
  • It sends the raw text to an API Gateway endpoint (/summarise-text).
  • A single Lambda function calls Amazon Bedrock (Titan) to create a five-point summary.
  • The summary is returned instantly and shown below.

Tech stack

  • AWS Lambda (Python 3.12)
  • Amazon Bedrock – Titan Text G1 Lite
  • Amazon API Gateway HTTP API
  • Static site hosted on GitHub Pages
