S3 Summariser

Upload a text file and get a five-bullet summary via AWS Lambda + Bedrock.

1. Upload a .txt file

2. Direct API call

POST the raw text to the endpoint:

POST https://wdz6s6z6vi.execute-api.us-east-1.amazonaws.com/summarise-text { "text": "Any long text you want summarised…" }

How it works

  • The browser reads your file locally.
  • It sends the text to /summarise-text on API Gateway.
  • Lambda calls Amazon Bedrock (Titan) to generate exactly five bullet points.
  • The summary is returned instantly and shown below.

Tech stack

  • AWS Lambda (Python 3.12)
  • Amazon Bedrock – Titan Text G1 Lite
  • Amazon API Gateway HTTP API (with CORS)
  • Static page hosted on GitHub Pages
