About this tool

This tool summarizes plain text files using Amazon Bedrock models (Claude 3.5 Haiku or Titan). You can upload your own `.txt` files or use example content. It works best for plain English text under 10,000 characters.

Limitations:

  • Only supports `.txt` files.
  • Maximum file size and length may be limited by Lambda runtime.
  • Some Bedrock models (like Haiku) may intermittently throw access or timeout errors.
